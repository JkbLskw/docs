= Swing Kit

:commercial-feature: Swing Kit
include::{articles}/_commercial-banner.asciidoc[opts=optional]

Swing Kit is a tool designed to render Vaadin applications inside Swing clients instead of standalone browsers.

The prime use case for this tool is to approach Swing to Vaadin modernization in a non-invasive way, which allows the codebase to be gradually transitioned into Vaadin upgrading single views.

== Requirements

- A server java appication with Vaadin Flow 23 <<{articles}/overview#,Flow>>;

- A desktop swing-based application running with at least Java 11;

- Maven build system.

Follow the <<setup#,Setup Guide>> to setup you swing project to integrate with a Vaadin Flow server.
